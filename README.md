# Année 1 - SAE 1.05
Projet Escape Game - SAE 1.05 | Année 1 | IUT Montpellier-Sète

# Descriptif
**Sujet** : Dans un contexte professionnel, un client demande à l'étudiant de formaliser ses attentes liées à un projet d'escape game. Cette SAÉ permet de se familiariser avec la conduite de projet à partir d'un besoin client. Ce projet mobilise 3 ressources :
- **R1.02 - Développement d'interfaces web** : Création d'un site internet (1 livrable).
- **R1.08 - Gestion de projet et des organisations** : Imaginer et formaliser un projet d'escape game + présentation orale (3 livrables).
- **R1.11 - Bases de la communication** : Création d'un poster / infographie (1 livrable).

L'ensemble du sujet est disponible en [pdf](Sujet%20SAE%201.05%20Escape%20Game.pdf).

_**Remarque** : Ce dépôt traîte uniquement de la création du site internet du projet d'escape game, reçu par une des équipes d'étudiants. Leur projet est disponible en pdf dans [resources/S6_L1_Doublet_Mounier_Dardalon_Winter_Pons.pdf](resources/S6_L1_Doublet_Mounier_Dardalon_Winter_Pons.pdf), ainsi que la conception des idées par notre équipe en pdf dans [resources/S1_L2_FRANCEUS_BINAMICKA_RIGAUX_KEARNS.pdf](resources/S1_L2_FRANCEUS_BINAMICKA_RIGAUX_KEARNS.pdf)._
